🔐 Midtrans Keys Validator

📋 Cara Mendapatkan Keys:
1. Login ke: dashboard.sandbox.midtrans.com
2. Klik Settings (⚙️) di sidebar kiri
3. Pilih Access Keys
4. Klik icon Copy (📋) di samping masing-masing key
5. Paste di form bawah ini
Format: SB-Mid-server-... (untuk sandbox)
Format: SB-Mid-client-... (untuk sandbox)
